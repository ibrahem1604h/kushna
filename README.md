# Kshna Restaurant Digital Menu

Static responsive menu website built from `menu_export.xlsx`.

## Files
- `index.html` main page
- `style.css` design and responsive layout
- `script.js` menu data from spreadsheet
- `app.js` search, category filter, Arabic/English toggle
- `assets/` uploaded brand references/logos

Open `index.html` in any browser or upload the folder to hosting.
